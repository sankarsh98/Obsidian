# Large Language Models (LLMs)

A **Large Language Model** is a deep learning algorithm that can recognize, summarize, translate, predict, and generate content using very large datasets.

## Key Definitions
-   **Token**: The basic unit of text (part of a word).
-   **Parameter**: The weights in the neural network (e.g., 7 Billion, 175 Billion).
-   **Context Window**: How much text the model can "see" at once.
-   **Inference**: The process of using the model to generate text.

## History
-   **2017**: Transformer Paper ("Attention Is All You Need").
-   **2018**: BERT (Encoder) & GPT-1 (Decoder).
-   **2020**: GPT-3 (The scaling hypothesis).
-   **2022**: ChatGPT (RLHF makes it usable).
-   **2023**: Llama (Open weights), GPT-4.
-   **2024**: Gemini, Claude 3 (Multimodality, Long Context).
