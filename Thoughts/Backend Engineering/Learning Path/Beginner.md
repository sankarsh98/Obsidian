# Beginner Track

> *Your first 12 weeks in Backend Engineering*

---

## Overview

| Duration | Focus | Outcome |
|----------|-------|---------|
| 12 weeks | Fundamentals | Entry-level ready |

---

## Curriculum

### Weeks 1-3: Programming
- [ ] Choose Python or Node.js
- [ ] Variables, functions, loops
- [ ] Data structures (arrays, objects)
- [ ] File I/O
- [ ] Error handling

### Weeks 4-6: Web & APIs
- [ ] HTTP protocol basics
- [ ] REST principles
- [ ] Build first API
- [ ] Postman/Insomnia testing
- [ ] Input validation

### Weeks 7-9: Databases
- [ ] PostgreSQL installation
- [ ] SQL basics (CRUD)
- [ ] JOINs and aggregations
- [ ] ORM (SQLAlchemy/Prisma)
- [ ] Database design

### Weeks 10-12: First Project
- [ ] Task Manager API
- [ ] User authentication
- [ ] Deploy to cloud
- [ ] Write documentation

---

## Checklist

- [ ] Complete 1 programming language course
- [ ] Build 3 small APIs
- [ ] Deploy 1 project live
- [ ] Understand SQL fundamentals
