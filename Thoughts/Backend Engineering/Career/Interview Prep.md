# Interview Preparation

> *Ace your backend engineering interviews*

---

## Interview Rounds

| Round | Focus |
|-------|-------|
| Phone Screen | Basics, motivation |
| Coding | Algorithms, problem-solving |
| System Design | Architecture |
| Behavioral | Experience, culture |

---

## Coding Questions

### Common Patterns
- Arrays and strings
- Hash maps
- Two pointers
- BFS/DFS
- Dynamic programming (basic)

### Practice
- LeetCode (Medium level)
- 2-3 problems per day
- Focus on explaining approach

---

## System Design Topics

1. **URL Shortener** - Basics
2. **Rate Limiter** - Algorithms
3. **Twitter Timeline** - Feeds
4. **Chat System** - Real-time
5. **Payment System** - Transactions

### Framework
1. **Clarify** requirements
2. **Estimate** scale
3. **Design** high-level
4. **Deep dive** components
5. **Discuss** trade-offs

---

## Behavioral (STAR Method)

- **S**ituation: Context
- **T**ask: Your responsibility
- **A**ction: What you did
- **R**esult: Outcome

### Sample Questions
1. Tell me about a challenging bug
2. How do you handle disagreements?
3. Describe a project you're proud of
