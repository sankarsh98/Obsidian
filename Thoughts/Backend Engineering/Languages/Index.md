# Languages Index

> *Backend programming languages*

## Choose Your First Language

| Language | Best For | Learning Curve |
|----------|----------|----------------|
| [[Python]] | Beginners, rapid development | Easy |
| [[JavaScript\|Node.js]] | Full-stack, real-time apps | Medium |
| [[Go]] | Performance, microservices | Medium |
| [[Java]] | Enterprise, large systems | Steep |

---

## Language Comparison

| Feature | Python | Node.js | Go | Java |
|---------|--------|---------|-----|------|
| **Typing** | Dynamic | Dynamic | Static | Static |
| **Performance** | Moderate | Good | Excellent | Good |
| **Concurrency** | Threading/Async | Event loop | Goroutines | Threads |
| **Ecosystem** | Huge | Huge | Growing | Massive |
| **Use Cases** | APIs, ML, Scripts | Real-time, APIs | Microservices | Enterprise |

---

## Recommendation

### Just Starting Out?
**Python** - Easy syntax, huge community, versatile

### Want Full-Stack?
**Node.js** - Same language frontend and backend

### Need Performance?
**Go** - Fast, simple, great for microservices

### Enterprise Work?
**Java** - Job market, established patterns
