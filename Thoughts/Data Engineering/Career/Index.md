## Job Search Tips

1. **Build projects** - 2-3 portfolio projects
2. **Get certified** - One cloud certification
3. **Network** - LinkedIn, meetups, conferences
4. **Apply strategically** - Quality over quantity
5. **Prepare deeply** - System design is key
