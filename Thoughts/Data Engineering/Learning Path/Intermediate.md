# Intermediate Learning Track

> *Level up your Data Engineering skills*

---

## Overview

| Duration | Prerequisites | Outcome |
|----------|---------------|---------|
| 16 weeks | Beginner track complete | Mid-level DE ready |

---

## Curriculum

### Weeks 1-4: Advanced SQL & Spark

**Advanced SQL**
- Query optimization and EXPLAIN plans
- Indexing strategies
- Advanced window functions
- Recursive CTEs
- Performance tuning

**Apache Spark**
- DataFrame API deep dive
- Spark SQL
- Transformations vs Actions
- Partitioning and caching
- Spark UI analysis

### Weeks 5-8: dbt & Data Quality

**dbt Mastery**
- Models and materializations
- Incremental models
- Tests and documentation
- Macros and packages
- dbt Cloud vs Core

**Data Quality**
- Great Expectations
- Soda checks
- Data contracts
- Monitoring and alerting

### Weeks 9-12: Cloud Deep Dive

**Pick ONE and go deep:**
- AWS: Glue, EMR, Redshift, Step Functions
- GCP: Dataflow, BigQuery, Composer
- Azure: Synapse, Databricks, Data Factory

**Certification prep** for your chosen platform.

### Weeks 13-16: Production Systems

**Production Readiness**
- CI/CD for data pipelines
- Testing strategies
- Logging and monitoring
- Alerting and on-call
- Documentation

**Project: Production Pipeline**
- Multi-source ingestion
- Complex transformations
- Data quality gates
- Automated testing
- Monitoring dashboard

- [ ] Build a Spark job processing 100GB+
- [ ] Create a dbt project with 20+ models
